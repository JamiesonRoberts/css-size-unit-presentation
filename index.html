What Atomic is not

Its not a framework, naming convention, boilerplate, etc.

Ex. - BEM, OOCSS, Bootstrap, Foundation

But it is influenced/influencing these.

What Atomic is

Atomic design is methodology for organizing and creating systems of components.

Forces you to think about complexity of components and what is actually influencing other elements.

Creates a layered approach to force structure, as well as create an easier bridge between dev and design with conversation instead of just hand-off.

Each layer should be standalone

Atoms

Like matter, basic building blocks of a website.

Doesn't mean you need to have a partial for each html element tag

Infact you probably shouldn't.

Grid, typography, forms, animations, normalize, etc

Deals with the interaction of atoms and creates a small standalone object

Why?

Way of structuring

Language independant while providing a way of structuring without spcific naming convetions

Normalizes file structure so that each type of technology being used could use the file same structure

Segmented code

Makes easier to segment code to prevent un-intended interactions either by code or by others programming on the same file

Especially useful for front-end technology like CSS

Defines levels of interactions

Each layer deals with specific interactions, and means it should be easier to trace down what is affecting what

EG. Organism level deals with the interaction of molecules and specific overrides

EG. Molecules deal with interactions of those specific atoms that are part of that component

Why this is 'better' than other methodologies?

Its not nessicarily, its your choice!

Each of them use various aspects of each other

Why do I like Atomic better than OOCSS, BEM, & SMACCS

You don't end up with class and id naming hell.

EG. .menu .menu__item .menu__item_state_current

You do end up with clearly defined levels of interaction and scoping of a varied state.

It doesn't have to be an all or nothing approach

There are numerous cross over pieces to the methodology puzzle, and Atomic can fit in, or stand alone.
